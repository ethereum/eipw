markdown-spell

Final section in proposal is the CC0-1.0 copyright waiver.

Examples

error[markdown-copyright]: the `Copyright` section must be the last content in the file
  |
8 | ## Copyright
  | ^^^^^^^^^^^^ nothing may follow this section
  |
  = help: end the file with `## Copyright` followed immediately by `Copyright and related rights waived via [CC0](../LICENSE.md).`, with no other content after it

Explanation

markdown-copyright ensures that the final section of proposals uses the correct copyright waiver. All proposals are required to be released under CC0-1.0 so that everyone is free to use and modify the standard in any way.
